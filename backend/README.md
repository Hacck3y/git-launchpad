# Git Launchpad Backend

FastAPI backend that handles deploying GitHub repos into Docker containers with AI-powered stack detection.

## Setup

```bash
pip install fastapi uvicorn
```

## Run

```bash
python main.py
# or
uvicorn main:app --host 0.0.0.0 --port 8000
```

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/deploy` | Start a deployment |
| GET | `/api/deploy/{id}` | Get deployment status |
| DELETE | `/api/deploy/{id}` | Kill a deployment |

## Deploy Request Body

```json
{
  "repo_url": "https://github.com/user/repo",
  "env_vars": { "KEY": "value" },
  "deploy_config": {
    "language": "Python",
    "framework": "FastAPI",
    "install_cmd": "pip install -r requirements.txt",
    "build_cmd": "",
    "start_cmd": "uvicorn app:app --host 0.0.0.0 --port 8000",
    "port": 8000,
    "dockerfile_content": "FROM python:3.12-slim..."
  }
}
```

The `deploy_config` field is generated by the AI analysis edge function and tells the deployer exactly how to build and run the project.
